# Should you learn to code in 2025?

The number one question I get asked is, "Should I learn how to code now that AI has made it so easy to build apps?" And I get it. You can build full enterprise applications now without writing a single line of code. I mean, I did it. I built an app that's now making over 300K a year, and I didn't write a single line of code inside of it. My answer is pretty simple. You don't have to, but you should. I have over 70 videos on my channel explaining how you can build apps without writing any code at all. And I truly believe you can build amazing apps that make you money with zero programming experience. But here's the thing, knowing how to program makes you a way better manager of AI agents. And in the near future, AI programming will be just managing tons of AI agents. I mean, we're already doing it now. Cursor has background agents that you just manage and it builds you the app. But part of being a great manager is understanding what your employees are doing. If you understand what they're doing better, you can give them better instructions. It's like a basketball coach trying to coach a player how to shoot a jump shot, right? Like the coach can say, "Hey, please put the ball on the net." But if they understand all the mechanics, where you put your elbow, where you put your wrist, you can give them a lot better instructions on how to shoot the basketball. But here's the thing. I don't think you need to take programming courses. I don't think you need to spend $200,000 on a degree. I don't think you need to do a whole bunch of tutorials on YouTube on how to program. The best way to learn how to code is just by doing. I download cursor, open it up, and start building. But here's the twist. As you're building, include in the prompt, explain how you built this and how the code works. Then after each step, read the description AI gives and look at the code it wrote. Even if you don't know what the code means, just read through it and take a look at the AI's description of what it does. It's the same thing as a child who learns how to speak English just by being around their English-speaking parents. Eventually, over time, you slowly learn what it means, and you're able to become fluent in that language. It's the same thing with coding. Just by taking a look at it, reading descriptions, over time, you'll get better and better, and your prompting is going to get way better, too, because you understand what the AI is doing. And the better your prompts get, the better the code the AI will write and the more money your app will end up making. So, should you learn how to code in the age of AI? The answer is yes. And the good news is you have a tutor built in to the AI you're using already. So, just change your prompting a little bit. Instruct the AI to teach you what it's doing as it goes along. And you'll learn a lot as you go. You don't need to be a senior developer now to build amazing apps.
